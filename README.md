#### 插件列表

##### 增强功能型
* **delimitMate**(自动加括号，换行等)
* **indentLine**(缩进线)
* **nerdcommenter**(注释插件)
* **emmet-vimEmmet**(前端开发神器)
* **vim-css-color**(CSS颜色显示)
* **vim-airline**(状态栏)
* **vim-airline-themes**(状态栏主题)
* **nerdtree**(查看项目文件)
* **deoplete**(自动补全神器)
* **neomake**(代码检错)
* **vim-autoformat**(代码格式化)
* **ultisnips**(代码块)
* **vim-snippets**(常用代码块集合)
* **vim-ctrlspace**(buffer,tab列表)
* **vim-mark**(高亮，看源码神器)
* vim-easymotion(快速跳转)

##### 语法
* **Dockerfile.vim**(Dockerfile语法支持)
* **vim-json**(JSON语法支持)
* **vim-pug**(Pug语法支持)
